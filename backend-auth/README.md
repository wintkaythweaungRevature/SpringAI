# Backend Auth - Member Registration

Add these files to your Spring Boot project for member login/signup.

**Files to add:**
- AuthController.java
- JwtService.java  
- SecurityConfig.java

**Dependencies (pom.xml):**
- spring-boot-starter-security
- jjwt-api, jjwt-impl, jjwt-jackson (0.12.3)

**Config:** Set `jwt.secret=your-32-char-secret` in application.properties

See README for full setup.
